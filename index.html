Reflection On Proposal

Landing Page

Mock Up

The changes made to the Landing Page was the remove of the circle in the centre of the main banner and the lines between the campaign photos. Feedback revealed that the circle would suggest that the user had to pull it down to extend the banner and the lines posed no purpose for the page. The arrow next to the donate button was removed because users were led to believe it would swipe to the right as a transition. The Solar Citizens logo also acts as a Landing Page link. The Solar Citizens logo in the header acts as a About Page Link where as the image and Stop the Sun Tax word are links to the Campaign Page. For the mobile version of the page, the text was removed from the campaign panel as it was too similar to the mobile Campaign page. The mobile version makes the campaigns salient since DesCo wanted to highlight the purpose of Solar Citizens - campaigning for renewable energy.

Website/Mobile

The circle seems like a pull tab and something will pop up.

About Page

Mock Up

The mock up desktop page was executed almost exactly the same besides the content which included a video but was placed in the same area as the image in the mock up. Users found the About Page on desktop easy to navigate and use. The mobile version of the website was slightly changed where the information was no longer placed on a panel as it restricted the content within. The circle of the header was also removed since it covered unnecessary space. An issue with the mobile version was the placement of the video had a margin-right that was unable to be fixed and caused it to be shifted far left.

Campaign Page

Mock Up

The Campaign Page was kept mostly the same with the exception changing of the signing button as Rob suggested to make the buttons bigger to make it more appealing for calls to action and to make the text different to indicate to the user that there are different petitions to sign. The mobile version was tried to be kept the same but there was many issues with trying to fix it as the containers and sections kept overlapping and the final execution was not usable.

Website/Mobile

Consider making the 'sign petition' button bigger so it is a more clear call to action. Might be better to say 'sign this petition' as well to make it clear there are multiple petitions.

Join Page

Mock up

The Join Page was kept the same with some minor additions to the page such as the petition checkboxes, which was linked from the campaigns page. DesCo wanted to provide a quick sign option so we connected it in the sign up form. The text within the sign up button was also changed to "I'M IN" since the current Solar Citizen's website also has that and we wanted to incorporate a feeling of participating in a greater good. After the user had pressed the button, they are prompted with a message saying "You're In!". The mobile version was relatively the same as the desktop counterpart but with small issues of alignment.

Website/Mobile

Donate Page

Mock Up

The Donate Page layout was kept the same but DesCo removed the breadcrumps navigation as we felt it was unnecessary. There would only be two sections for the breadcrumps, which led to the decision to remove the breadcrump navigation overall. The mobile version was kept the same. There was an added feature to the page where the user would be prompted with a message saying 'Thank you!' to alert the user it worked.

Website/Mobile

Navigation

Mock Up

The navigation bar was kept the same for all desktop pages, however for the mobile, it had to be changed due to the Join and Donate buttons not being able to be placed in the list with the hamburger menu. To solve the issue, DesCo created a new page that resembled a menu for mobile users. The hamburger menu icon was removed and replaced with the text, 'Menu', to make it simple and understanding.

Website/Mobile

Issues

The main issuses revovle around the web responsiveness to mobile usage. These issues include:

  • The video on the About Page had issues centering on mobile.
  • The panels on the Campaign Page on the mobile did not align with one another and kept overlapping at 480px but at 767px it functioned nicely.
  • The alignment of the checkboxes on the Join Page was not straight.
  • The mobile navigation caused a lot of issues due to the Join and Donate buttons not being in a list along with the About and Campaigns links. This prevented the hamburger menu being applied to the mobile version.

Future works aims for DesCo to learn and fix these issues and refine the website more.
